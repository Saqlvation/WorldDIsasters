# WorldDIsasters
A website that shows you the map of all the ongoing natural disasters like wildfires, storms, volcanoes ecc..


### NASA's API
for this project i used NASA' seonet API fetched from [eonet](https://eonet.gsfc.nasa.gov/), you can look it up yourself and find all the documentation and your free API, EONET uses GeoJSON format so when we are gonna look at the coordinates( once fetched, which is how i made it find all the events ) it will be LONGITUDE,LATITUDE in that order.
